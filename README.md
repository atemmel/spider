# spider

Keybinds:

```
'h': Left (../) ←
'l': Right (go down/open) →
'j': Down ↓
'k': Up ↑
'c': Create file
'C': Create directory
'D': Delete item(s)
'f': Fast travel
' ': Mark item
'R': Rename item
'm': Remove all marks
'p': Paste marked files
'v': Move marked files
```

## Dependencies

* curl
* ncurses
* tidy
* zig 0.11.0
