# spider (zig branch)

Use at your own risk :^)

```
'h': Left (../) ←
'l': Right (go down/open) →
'j': Down	↓
'k': Up ↑
'c': Create file
'C': Create directoru
'D': Delete item(s)
'f': Fast travel
' ': Mark item
'R': Rename item
'G': Enter git mode (experimental)
'm': Remove all marks
'a': Read MIME data
'p': Paste marked files
'v': Move marked files
```
